# imserver
 A http file server for image/markdown/python-code 

## Install
pip install https://github.com/ddn5/imserver.git

## Install_depands
 - [bottle]()
 - [markdown]()
 - [paste]()

## Third-party modules
 - PyFontify (from pyqt pkg)
 - py2html (from pyqt / modified)

## Usage
python -m imserver <port>
